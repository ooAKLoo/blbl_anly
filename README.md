# UPulse

B 站 UP 主视频数据分析与可视化桌面应用，帮助你从数据角度发现有趣的内容趋势和规律。

## 功能

- 输入 UP 主 MID，自动获取公开视频数据
- 多维度数据可视化（播放趋势、发布规律、时长分布等）
- 跨 UP 主对比分析
- 支持 macOS / Windows / Linux
