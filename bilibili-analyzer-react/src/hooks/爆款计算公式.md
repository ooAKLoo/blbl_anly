## 业界常用的异常点检测方法

### 1. IQR 四分位距法（推荐）

箱线图背后的统计方法，对长尾分布更稳健：

```javascript
// 播放量通常是长尾分布，IQR 比均值更稳健
const sorted = [...playCountsyya].sort((a, b) => a - b);
const Q1 = sorted[Math.floor(sorted.length * 0.25)];
const Q3 = sorted[Math.floor(sorted.length * 0.75)];
const IQR = Q3 - Q1;

const upperBound = Q3 + 1.5 * IQR;  // 标准异常值
const extremeBound = Q3 + 3 * IQR;  // 极端异常值（爆款）
```

### 2. Z-Score 标准分法

```javascript
const mean = plays.reduce((a, b) => a + b, 0) / plays.length;
const std = Math.sqrt(plays.reduce((sum, p) => sum + (p - mean) ** 2, 0) / plays.length);

// Z > 2 是显著异常，Z > 3 是极端异常
const zScore = (playCount - mean) / std;
const isViral = zScore > 2;
```

### 3. 对数空间 Z-Score（最适合播放量）

播放量是典型的对数正态分布，取 log 后再算 Z-Score 更准确：

```javascript
const logPlays = plays.map(p => Math.log10(p + 1));
const logMean = logPlays.reduce((a, b) => a + b, 0) / logPlays.length;
const logStd = Math.sqrt(logPlays.reduce((sum, p) => sum + (p - logMean) ** 2, 0) / logPlays.length);

const logZ = (Math.log10(playCount + 1) - logMean) / logStd;
const isViral = logZ > 1.5;  // 对数空间 1.5 相当于线性空间的 ~3 倍
```

## 方法对比

| 方法 | 查准率 | 查全率 | 适用场景 |
|------|--------|--------|----------|
| **均值 N 倍** | 低 | 中 | 快速粗筛 |
| **IQR** | 高 | 高 | 通用稳健 |
| **Z-Score** | 中 | 中 | 正态分布数据 |
| **对数 Z-Score** | 高 | 高 | 播放量等长尾数据 ✅ |

## 推荐方案

```javascript
const detectViralVideos = (growthData) => {
  if (growthData.length < 5) return []; // 数据太少不判断

  const plays = growthData.map(d => d.playCount);
  
  // 对数空间计算
  const logPlays = plays.map(p => Math.log10(Math.max(p, 1)));
  const logMean = logPlays.reduce((a, b) => a + b, 0) / logPlays.length;
  const logStd = Math.sqrt(
    logPlays.reduce((sum, p) => sum + (p - logMean) ** 2, 0) / logPlays.length
  );

  const viralThreshold = 1.5; // 可调参数

  return growthData
    .map((d, i) => ({
      ...d,
      index: i,
      logZ: (Math.log10(Math.max(d.playCount, 1)) - logMean) / (logStd || 1)
    }))
    .filter(d => d.logZ > viralThreshold && d.playCount > 10000) // 相对+绝对阈值
    .sort((a, b) => b.logZ - a.logZ); // 按爆款程度排序
};
```

这样能识别出**所有显著高于该 UP 自身水平的视频**，而不是只取一个最大值。
